## ⭐ 핵심 변화 TOP 3

### 1. Claude Code 실비용 공개 논란 — 하루 $13, 상위 10%는 $30

**📌 무슨 일?**

Anthropic의 Claude Code 공식 문서에 기재된 "평균 하루 사용 비용"이 조용히 바뀌었어.

변화 전후를 표로 정리하면:

| 항목 | 이전 (4월 16일 전) | 현재 |
|---|---|---|
| **개발자 1인 하루 평균 비용** | $6 | **$13** |
| **90% 상한선(하루)** | $12 | **$30** |
| **월 추정 비용** | ~$90 | **$150 ~ $250** |

4월 29일 TechBriefly, Let's Data Science 등 여러 미디어가 이 변화를 동시에 보도하면서 주목을 받았어.

이전에 Simon Willison(웹 개발 유명 블로거)이 4월 22일 "Claude Code가 월 $100 드나?"라는 글로 처음 이 이슈를 제기했고, 4월 29일 대형 미디어로 번졌어.

**🤔 왜 중요?**

두 가지 이유에서 중요해:

1. **투명성 문제** — Anthropic이 공식 문서를 업데이트했지만 별도 공지 없이 조용히 바꿨어. 사용자들은 청구서를 보고서야 인지하는 경우가 많았어.

2. **기업 도입 계산 변화** — 개발자 10명이 한 달 풀가동하면 월 $1,500~$2,500. 이전 추정($900)과 크게 달라져. 기업이 ROI를 계산할 때 전제가 바뀐 거야.

비유하자면 — "택배 기본요금은 3,000원이라더니 실제로 보내봤더니 평균 6,500원"이라는 고지를 나중에 받은 것과 같아.

**👉 너에게 영향**

- **Claude Code 사용자라면**: `~/.claude/settings.json`에서 `CLAUDE_CODE_TOKEN_BUDGET`을 설정하면 하루 토큰 한도를 직접 제어할 수 있어. 예: 하루 $10 제한을 걸고 싶다면 현재 토큰 단가($15/M input 기준)로 역산.
- **무료 원칙 재점검**: Claude.ai Pro($20/월) 플랜 내에서 Claude Code를 쓰는 방식이라면 이 추가 비용이 발생하지 않아. API 직접 사용자만 해당되는 이야기야.
- **모니터링 권장**: Anthropic이 비용 추정치를 조용히 올릴 수 있다는 선례가 생겼어. 청구 알림 설정을 평소보다 낮게 잡아두는 게 좋아.

### 2. Anthropic 런던 800인 오피스 + OpenAI King's Cross — "영국 AI 러시"

**📌 무슨 일?**

4월 29일 Bloomberg가 영국 AI 오피스 임대 동향을 단독 보도했어. 주요 내용:

**Anthropic:**
- 런던 Euston의 **1 Triton Square** 빌딩에 **158,000 sq ft(약 14,700㎡)** 임차 확정
- 현재 영국 인력의 **4배** 규모인 800명 수용 가능
- 이전 4월 16일 발표(800인 계획)를 실제 계약으로 공식화한 것

**OpenAI:**
- King's Cross **Regent Quarter**에 **77,000 sq ft** 임차
- 544석 규모, 2027년 오픈 예정
- 영국 첫 번째 영구 오피스

**런던 AI 임대 빅픽처:**
- 2025년 초부터 AI 관련 기업들이 런던에서 **100만 sq ft 이상**을 임차
- 그 중 절반 이상이 **4월 한 달**에 집중됨
- Bezos의 Project Prometheus도 King's Cross 사무실을 검토 중 [unverified]

**🤔 왜 중요?**

AI 기업들의 런던 집중은 단순한 오피스 이전이 아니야:

1. **미국 정치 리스크 헤징** — Pentagon 공급망 리스크 지정, 트럼프 행정부와의 마찰 → 유럽 거점이 "보험"이 돼
2. **EU/UK 규제 대응** — EU AI Act, UK AI 규제 논의에 현지 조직이 없으면 대응이 느려
3. **글로벌 인재 유치** — 런던은 미국 외 AI 인재 밀도가 가장 높은 도시 중 하나

**👉 너에게 영향**

- **단기 영향 없음**: 개인 사이드 프로젝트에는 직접 영향 없어.
- **중장기 신호**: Anthropic이 영국/유럽 시장에 공격적으로 투자하고 있다는 것 = UK/EU 사용자 대상 서비스(응답 속도, 데이터 레지던시)가 개선될 가능성.

### 3. Pentagon vs. 백악관 — Anthropic "공급망 리스크" 공방 업데이트

**📌 무슨 일?**

4월 29일 CNBC가 이 사건의 새로운 국면을 보도했어. 시간순으로 정리하면:

| 날짜 | 사건 |
|---|---|
| **2026-02-27** | 트럼프 대통령, 연방 기관 Anthropic AI 사용 중단 지시 |
| **2026-03-05** | 국방부(Pentagon), Anthropic을 "공급망 리스크"로 공식 지정 |
| **2026-03-09** | Anthropic, 트럼프 행정부 상대로 소송 제기 |
| **2026-03-26** | 연방 판사, 공급망 리스크 지정에 임시 금지 명령 발동 |
| **2026-04-29** | 백악관, 연방 기관이 Anthropic 사용할 수 있도록 우회 방안 초안 작성 중 (CNBC 보도) |

갈등의 근본 원인:

- **국방부 요구**: Claude를 완전 자율 무기, 국내 대규모 감시에도 사용 허용해줘.
- **Anthropic 입장**: 그 두 용도는 우리 이용약관 위반이야. 다른 건 다 좋아.

트럼프 대통령이 최근 CNBC 인터뷰에서 Anthropic이 "잘 하고 있고, 앞으로 크게 쓸 수 있다"고 말하면서 관계 회복 신호를 보냈어.

**🤔 왜 중요?**

Anthropic의 미국 연방 정부 매출은 수백억 달러 규모 잠재 시장이야. 이 공방의 결론에 따라:

- **화해 시나리오**: 정부가 Claude를 다시 쓸 수 있게 되면 Anthropic 수익 기반이 빠르게 확대돼.
- **장기 마찰 시나리오**: AI 안전 원칙과 정부 계약 사이의 긴장이 AI 산업 전체 선례가 될 수 있어.

**👉 너에게 영향**

- **직접 영향 없음**. 하지만 이 사건은 "AI 기업이 정부 고객과 어떤 원칙을 지켜야 하는가"라는 역사적 선례를 만드는 중이야.

## 🔶 Anthropic / Claude Code 깊이 분석

### Claude Code v2.1.122 / v2.1.123 상세

두 버전이 이틀 연속 출시됐어:

| 버전 | 출시 | 핵심 변화 |
|---|---|---|
| **v2.1.122** | Apr 28 | Bedrock 서비스 티어 선택, PR URL → 세션 찾기, `/mcp` 중복 경고, OpenTelemetry 강화 |
| **v2.1.123** | Apr 29 | OAuth 401 인증 루프 버그 수정 |

**v2.1.122 상세 설명:**

1. **Bedrock 서비스 티어 환경변수** — `ANTHROPIC_BEDROCK_SERVICE_TIER=flex` 또는 `=priority`로 AWS Bedrock 호출 시 비용 vs. 속도 우선순위를 선택 가능. Flex는 저렴하지만 느리고, Priority는 빠르지만 비쌈. 4월 27일 Gemini API에 추가된 것과 같은 개념이 AWS Bedrock에도 생긴 거야.

2. **PR URL → 세션 바로 찾기** — Claude Code의 `/resume` 명령에서 GitHub, GitLab, Bitbucket PR URL을 붙여넣으면 해당 PR을 만든 세션을 바로 찾아줘. 이전엔 날짜나 브랜치명으로 찾아야 했는데, 이제 "이 PR 만들 때 무슨 맥락이었지?" 한 번에 복구 가능.

### Anthropic 비용 정책 종합

4월 16일 이후 조용히 업데이트된 Claude Code 비용 추정치는 사실 Anthropic의 **실사용 데이터 기반 업데이트**야.

즉, "올렸다"기보다는 "실제 사용하는 개발자들의 평균이 높아져서 추정치를 현실화한 것"이라는 해석도 있어. Claude Code의 에이전트 기능(sub-agents, 자동 수정, 테스트 실행)이 고도화될수록 자연히 토큰 소비가 늘어나거든.

## 🔬 트렌드 한 컷: AI 기업들의 "런던 러시" — 왜 지금인가?

2025년 초부터 AI 기업들이 런던에서 100만 sq ft 이상을 임차했고, 그 절반이 4월 한 달에 몰렸어.

**비유**: 1990년대 닷컴 붐 때 실리콘밸리 오피스 임대료가 수직 상승했던 거 기억해? 지금 런던이 그 상태야. 단, 이번엔 실리콘밸리가 아니라 **런던 King's Cross 일대**가 중심이야.

**왜 런던인가?**

세 가지 이유가 겹쳤어:

1. **미국 정치 리스크 분산** — Pentagon 분쟁, 트럼프 행정부와의 갈등 → 유럽에 백업 거점 확보가 시급해짐
2. **EU AI Act 대응** — 2026년부터 EU AI Act가 본격 시행되면 현지 조직이 없는 기업은 규제 대응이 어려워져
3. **인재 지리학** — 런던은 전 세계 AI 박사 인력 밀집 도시 2위 (1위 샌프란시스코). 비자 장벽도 미국보다 낮아.

**이게 왜 중요한가?**

AI 인프라가 지금 "샌프란시스코 독점"에서 **"샌프란·런던 양극 체제"**로 재편되는 초기 신호야.

역사에서 비슷한 패턴: 인터넷 초기엔 미국 독점이었다가, 2010년대에 유럽 스타트업 생태계가 런던·베를린·스톡홀름 중심으로 뿌리내렸어. AI는 그 전환이 훨씬 빠르게 일어나고 있어.
